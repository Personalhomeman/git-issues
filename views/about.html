What does this do?

It's hard being a maintainer/active contributor of an open source project üò≠. You have to keep an eye on all the bug reports and pull requests and provide your review on the same. I have reliazed that I have been wasting so much of my time in browsing all the issues/PRs lately. The current flow involves going on an issue page and open all the issues in new tab in order to see the issue details etc.
To make this process a little bit easier, I have built this small tool. Here you don't need to open the issues in a new tab. In a single page you can browse all the issues/PRs and comment on the same. This saves a lot of time + RAM also üòõ.
